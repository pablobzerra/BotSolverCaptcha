Documentação da API

Endpoint: /api/solution/solver

Faz upload de um captcha para ser resolvido.

Exemplo de uso com Python:

import requests

url = 'http://solve2earn.ddns.net/api/solution/solver?token=12345'
files = {'captcha': open('captcha.png', 'rb')}
response = requests.post(url, files=files)
print(response.json())

Exemplo de uso com cURL:

curl -X POST -F 'token=12345' -F 'captcha=@captcha.png' http://solve2earn.ddns.net/api/solution/solver

Exemplo de uso com JavaScript:

// Exemplo usando o método fetch
const url = 'http://solve2earn.ddns.net/api/solution/solver?token=12345';
const fileInput = document.querySelector('#captchaFile');
const formData = new FormData();
formData.append('captcha', fileInput.files[0]);

fetch(url, {
  method: 'POST',
  body: formData
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));
